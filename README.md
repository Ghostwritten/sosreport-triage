# sosreport-triage

Lightweight skill for offline analysis of customer-provided `sosreport` bundles.

It is meant for support engineers who need to judge whether host resource pressure likely contributed to an incident without accessing the customer's production environment.

Core files:

- `SKILL.md`: the skill

Design goals:

- lightweight and low-token
- evidence-first and path-cited
- cautious about causality
- useful for support-style incident analysis
